#  Pulse App

**Live Demo:** https://pulse-app-store.netlify.app

## Description

**Pulse App** is a modern web application designed to showcase and explore apps in a sleek, responsive interface — similar to an app store experience.  
Users can browse different apps, view details, and interact with a clean and dynamic UI built for smooth navigation and performance.

This project demonstrates strong front-end development skills, UI/UX design principles, and API/data handling techniques.


## Features

- Interactive and responsive app listing interface
- Dynamic app details display
- Smooth navigation and transitions
- Modern design using Tailwind CSS
- Optimized for all screen sizes


## Technologies Used

- **React.js** – Front-end framework for building UI components
- **Tailwind CSS** – Utility-first CSS framework for styling
- **JavaScript (ES6+)** – Logic and interactivity
- **React Router** – For routing and navigation
- **Netlify** – For hosting and deployment
